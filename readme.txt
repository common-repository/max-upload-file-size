Increase Max Upload size limit for any kind of file. 


== Description==
Increase Max Upload filesize limit through dashboard directly.
Plugin automatically detects upload limits set by WordPress and set by the hosting provider, and displays them.


== Installation ==

* Upload the Max Upload File Size plugin to the `/wp-content/plugins/` directory
* Activate the Max Upload File Size plugin through the 'Plugins' menu in WordPress
* You can use it from WordPress menu 'Max Upload File Size' to change upload size.


== Frequently Asked Questions ==

= How to use plugin? =
* After activating plugin, go to WordPress dashboard and you will see menu on left 'Max Upload File Size'.

= Does this plugin work with all servers and hosting providers? =
* Yes, it works with all servers and hosting providers. However, please know that server adjusted limits can't be changed from a WordPress plugin. If server limit is lower then you have to email your hosting provider and ask them to increase the limit.
